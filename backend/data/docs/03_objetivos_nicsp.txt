# OBJETIVOS PRINCIPALES DE LAS NICSP

OBJETIVO GENERAL: Mejorar la calidad y utilidad de la información financiera del sector público a nivel mundial.

OBJETIVOS ESPECÍFICOS:

1. MEJORAR LA TRANSPARENCIA:
   - Información clara y accesible sobre el uso de recursos públicos
   - Visibilidad completa de la gestión financiera gubernamental
   - Divulgación de todas las operaciones significativas

2. FORTALECER LA RENDICIÓN DE CUENTAS:
   - Demostrar el uso responsable de fondos públicos
   - Mostrar el cumplimiento de mandatos legales
   - Facilitar la evaluación ciudadana de la gestión

3. PERMITIR COMPARABILIDAD:
   - Entre diferentes períodos contables de una misma entidad
   - Entre diferentes entidades públicas del mismo país
   - Entre gobiernos de diferentes países

4. FACILITAR TOMA DE DECISIONES:
   - Información robusta para formulación de políticas públicas
   - Base técnica para asignación eficiente de recursos
   - Análisis de costo-beneficio de programas gubernamentales

5. PROMOVER CONFIANZA:
   - Confianza ciudadana en las instituciones públicas
   - Credibilidad internacional para acceso a financiamiento
   - Base para una gobernanza económica sólida

BENEFICIOS DIRECTOS DE LA IMPLEMENTACIÓN:
- Mayor transparencia fiscal y gestión pública
- Mejores condiciones para financiamiento internacional
- Fortalecimiento de los sistemas de control interno
- Armonización con estándares internacionales
- Información para evaluación de sostenibilidad fiscal
